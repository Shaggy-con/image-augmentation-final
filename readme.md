# Image Augmentation Web Application

Developed a full-stack web application for image augmentation using Flask and Python, enables generation of diverse images for training AI models.

## Features
• Implemented various image transformations: rotation, scaling, flipping, brightness, contrast, saturation, and grayscale.
• Enabled asynchronous image upload and transformation using JavaScript and AJAX
• Integrated PIL (Python Imaging Library) for efficient image processing and augmentation

## Steps to run the project

Download the source folder

### To run backend
1. cd Image-augmentation
2. create virtual environment (python -m venv venv)
3. Run (pip install -r requirements.txt)
4. python app.py

### To run frontend
1. cd augment_frontend
2. Run (npm i)
3. npm run dev

Ensure .env files is available
